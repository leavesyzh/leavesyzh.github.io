表格布局检测与数据提取解决方案

一、方案逻辑

（一）表格图像预处理

对表格图像进行预处理，以增强表格的可见性并减少干扰。

（二）表格布局检测

利用 Layout-Parser 库中的 Detectron2LayoutModel 模型，PaddleOCR 的 PP-Structure 或者 DocLayout-YOLO 对预处理后的图像进行表格布局检测，识别出表格区域。

（三）表格识别与数据提取

将检测到的表格区域裁剪出来，然后使用开源的 OCR 工具（如 Tesseract-OCR）对裁剪后的表格区域进行文字识别，提取出表格中的数据内容。

（四）数据整理与保存

将提取到的表格数据整理为合适的数据结构，再利用 Pandas 库将其保存为 Excel 文件。

二、环境搭建

因为 Detectron2 并不支持 Windows 系统，使用 Docker 搭建工作环境。

  1. 基础配置：以 python:3.8-slim-buster 镜像为基础，设置非交互模式并创建工作目录。
  2. 创建工作目录：创建并切换到 /app 目录，作为后续操作的工作目录。
  3. 系统依赖安装：更新系统包列表，安装必要的系统依赖（如 HTTPS 支持、证书、编译工具、Git、OCR 工具等），并清理缓存。
  4. 源切换：将 APT 源切换为 HTTPS，使用官方安全仓库。
  5. Python 依赖安装：升级 pip，安装指定版本的 PyTorch、torchvision、Pillow、LayoutParser 及其 OCR 依赖，以及从源安装 Detectron2。
  6. 项目文件复制：将当前目录下的项目文件复制到容器的 /document-layout-detection 目录。
  7. 使用 VSCode 远程连接配置的 Docker 容器进行开发和测试
